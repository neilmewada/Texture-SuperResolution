# Texture-SuperResolution

## Dataset

Download the dataset and create a folder called "dataset" in the root of the repositor, and extract the contents of this zip file:

[Dataset.zip](https://drive.google.com/file/d/1i0awMsHx5ybMq1JLyr3tR3D2yMk57PXn/view?usp=drive_link)

The directory structure should look like this:
```
dataset
├── test
│   ├── x2
│   └── x4
├── train
│   ├── ...
│   ├── ...
│   └── ...
├── validation
```
